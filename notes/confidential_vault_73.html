Confidential Vault


Sometimes you need to store confidential information, but need to make sure nobody can detect your password:



The confidential vault allows you to enter ANY PASSWORD, and uses this to code the information you enter.



It does not store the password anywhere, but instead applies the password you enter to de-code the information.


If you enter a different password, the information is de-coded incorrectly:



This means even brute-force de-coding methods are no use as there is no way of knowing if the information has been decoded correctly.


You can even use a different password for each confidential vault.
